# ✈️ High Cloud Airlines Data Analysis

An end-to-end Data Analytics project that analyzes airline operations using **Excel, MySQL, Power BI, and Tableau** to answer key business questions and build interactive dashboards.

---

## 📌 Project Overview

The objective of this project is to analyze airline performance data, identify business insights, and visualize important KPIs through multiple analytics tools.

This project demonstrates a complete analytics workflow from raw data to business dashboards.

---

## 🛠️ Tools & Technologies

- Microsoft Excel
- MySQL
- Power BI
- Tableau 
- Git & GitHub

---

## 📂 Project Workflow

```
CSV Dataset
      │
      ▼
Excel Data Cleaning & Dashboard
      │
      ▼
Import Data into MySQL
      │
      ▼
SQL Business Analysis
      │
      ▼
Connect MySQL to Power BI
      │
      ▼
Interactive Power BI Dashboard
      │
      ▼
Recreate Dashboard in Tableau
```

## 📊 Project Stages

### ✅ Stage 1 – Excel Dashboard

- Data Cleaning
- Pivot Tables
- Pivot Charts
- KPI Cards
- Interactive Slicers
- Dashboard Creation

Dashboard Highlights

- Total Flights
- Total Passengers
- Load Factor
- Top Airlines
- Monthly Trends
- Route Analysis

---

### ✅ Stage 2 – SQL Analysis

The airline dataset was imported into MySQL for business analysis.

Business questions solved include:

- Yearly Passenger Trend
- Monthly Load Factor
- Top Airlines by Passengers
- Top Routes
- Distance Group Analysis
- Carrier Performance
- Flight Trends
- Passenger Distribution

SQL concepts used:

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE
- Date Functions
- Common Table Expressions (CTE)
- Window Functions

---

### ✅ Stage 3 – Power BI Dashboard

Connected the MySQL database directly to Power BI.

Developed an interactive dashboard using:

- KPI Cards
- Bar Charts
- Line Charts
- Slicers
- Filters
- Drill Down
- Interactive Visualizations

---

### 🚧 Stage 4 – Tableau Dashboard

Currently recreating the Power BI dashboard in Tableau.

Features being developed:

- KPI Dashboard
- Interactive Filters
- Trend Analysis
- Maps
- Performance Dashboard
